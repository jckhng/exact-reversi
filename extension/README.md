# Exact Reversi Extension Files

Copy these files into your Kindle extension folder:

- `config.xml` -> `/mnt/us/extensions/exact-reversi/config.xml`
- `menu.json` -> `/mnt/us/extensions/exact-reversi/menu.json`
- `launch_exactreversi.sh` -> `/mnt/us/extensions/exact-reversi/launch_exactreversi.sh`
- `stop_exactreversi.sh` -> `/mnt/us/extensions/exact-reversi/stop_exactreversi.sh`
- `tail_log_exactreversi.sh` -> `/mnt/us/extensions/exact-reversi/tail_log_exactreversi.sh`

Optional document shortcut:

- `shortcut_exactreversi.sh` -> `/mnt/us/documents/shortcut_exactreversi.sh`

KUAL is the reliable tap-launch path. The stock Kindle home screen normally
does not execute `.sh` files directly.
