# Kindle Iagno Extension Files

Copy these files into your Kindle extension folder:

- `config.xml` -> `/mnt/us/extensions/kindle-iagno/config.xml`
- `menu.json` -> `/mnt/us/extensions/kindle-iagno/menu.json`
- `launch_kindleiagno.sh` -> `/mnt/us/extensions/kindle-iagno/launch_kindleiagno.sh`
- `stop_kindleiagno.sh` -> `/mnt/us/extensions/kindle-iagno/stop_kindleiagno.sh`
- `tail_log_kindleiagno.sh` -> `/mnt/us/extensions/kindle-iagno/tail_log_kindleiagno.sh`

Optional document shortcut:

- `shortcut_kindleiagno.sh` -> `/mnt/us/documents/shortcut_kindleiagno.sh`

KUAL is the reliable tap-launch path. The stock Kindle home screen normally
does not execute `.sh` files directly.
